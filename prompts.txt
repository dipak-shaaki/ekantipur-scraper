Prompts used during development — ekantipur-scraper

PROMPT 1 — Article extraction skeleton

I inspected ekantipur.com/entertainment with DevTools (F12).
Each article is inside div.category > div.category-inner-wrapper.
Structure:
  - Title: .category-description h2 a (text)
  - Author: .author-name a (text, sometimes missing)
  - Image: .category-image img (src attribute)

Write a Playwright Python function that loops through div.category
elements, skips divs with no title (date separators), and extracts
title, image_url, author from top 5. Return null if author missing.


PROMPT 2 — Fixing null image URLs

image_url returns null for all articles. Debugged with
print(card.inner_html()) and found two patterns:

Pattern 1: <img class="loaded" src="https://assets-cdn-api...">
Pattern 2: <img class="lazy" data-src="https://assets-cdn-api...">

Current broken line:
img_el = card.query_selector(".category-image img.lazy")
image_url = img_el.get_attribute("data-src") if img_el else None

Fix to handle both — try data-src first, fall back to src.

Write a Playwright function to extract title, image_url, author
from the active slide. Return null for any missing field.
